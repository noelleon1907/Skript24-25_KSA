Cloud

Was ist eine Cloud?

Eine Cloud ist ein Netzwerk von Servern, welche über die ganze Welt verteilt sind, und miteinander über das Internet verbunden sind.

Eine Cloud kann verschiedene Aufgaben durchführen. Sie kann bspw. Dateien speichern, ein Webmail bereitstellen und Online-Office Dienste ausführen.

Mit welcher Cloud ist mein Gerät verbunden.

Mein Schul-Laptop ist mit dem 'One Drive' von Microsoft verbunden. Der Kanton Luzern stellt den Kantonsschulen während der Schulzeit 1 TB Cloudspeicher und die Office Anwendungen von Microsoft für Schul-Zwecke zur Verfügung.

Welche Verzeichnisse werden synchronisiert?

Auf meinem Laptop werden die Verzeichnisse 'Dokumente', 'Bilder' und 'Desktop' synchronisiert. Ausserdem wird alles synchronisiert, was im Ordner ☁︎ Noel - sluz, gespeichert wird.

Wo werden die synchronisierten Daten gespeichert?

Die Daten werden im Onedrive, im Datenzenter von Microsoft in der Schweiz gespeichert.
